# OLED Lyrics Animation

A CircuitPython project for the Raspberry Pi RP2040 that displays song lyrics on a 0.96" OLED display with a falling text animation. Each lyric line scrolls from the top to the bottom of the screen, allowing the animation speed to be synchronized with a song.

## Features

- Falling text animation.
- Automatic text wrapping.
- Centered text rendering.
- Adjustable animation speed for each lyric.
- Designed for synchronization with music.
- Written in CircuitPython.

## Hardware

- Raspberry Pi RP2040
- 0.96" OLED Display (128×64, I2C)
- Breadboard
- Jumper Wires

## Software

- CircuitPython
- Visual Studio Code
- Adafruit CircuitPython Libraries

## Library Dependencies

This project requires:

- `adafruit_ssd1306`

The following modules are included with CircuitPython:

- `board`
- `busio`
- `time`

## Wiring

| OLED Pin | RP2040 GPIO |
|----------|-------------|
| SDA | GP0 |
| SCL | GP1 |
| VCC | 3.3V |
| GND | GND |

## How to Run

1. Install CircuitPython on the RP2040.
2. Copy `code.py` to the `CIRCUITPY` drive.
3. Install the required library inside the `lib` folder.
4. Connect the OLED display using the wiring table above.
5. Edit the `LYRICS` list to include your own text and animation timing.
6. Reset the board.
